## ğŸ¨ Customization

### Changing Colors

Edit the color scheme in `styles.css`:

```css
/* Main color palette */
--primary-color: #1a1a1a;      /* Dark black */
--secondary-color: #f5f5dc;    /* Beige */
--accent-color: #d4c5a9;       /* Light brown */
```

### Updating Content

1. **Logo/Title**: Edit the `<h1>` tag in each HTML file's header
2. **Navigation**: Modify the `<nav>` section
3. **Sections**: Update content within `<section>` tags
4. **Contact Info**: Change details in contact.html

### Adding New Pages

1. Create a new HTML file (e.g., `gallery.html`)
2. Copy the structure from an existing page
3. Add a link in the navigation menu of all pages:
   ```html
   <a href="gallery.html">Gallery</a>
   ```
